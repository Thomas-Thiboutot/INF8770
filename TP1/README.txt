Packages à importer:
    - numpy
    - time
    - pillow (Python Imaging Library fork)
    - anytree

Pour exécuter l'encodage LZW sur les fichiers, exécutez le fichier LZW_encoding.py
Pour exécuter l'encodage Huffman sur les fichiers, exécutez le fichier huffman_encoding.py
Les performances des encodages LZW et Huffman seront enregistrées respectivement dans les fichiers
LZW_results.txt et Huffman_results.txt

SYNOPSIS         

       [algo].py [OPTION]...

DESCRIPTION         top

        Compresses files.

        -s, --print-statistics
              Shows statistics(compression rate, execution time)
              
        -h, --help display this help and exit
